## Introduction

**Audience Atlas** is a data-driven tool designed to help you analyze your GitHub connections. By mapping your followers and following lists, it identifies relationship gaps, such as users you follow who do not follow you back. 

Audience Atlas provides the insights you need to manage your GitHub network.

---

## Key Features

*   **Relationship Audit:** Instantly find non-reciprocal follows.
*   **Graph Visualization:** View your network as an interactive map via the web interface.
*   **Deep Insights:** Identify mutual followers and hidden network clusters.
